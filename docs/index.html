How It Works

From parameters to PDF in three steps

No account needed. No cloud dependency. Start the backend, open the UI, get results.

01
Select Region & Calculator
Choose from GCC, Europe/UK, India, or Australia. Pick any of the 26 modules. Design codes, correction factors, and compliance limits load automatically.
02
Enter Your Parameters
Input load data, dimensions, or system details. The UI guides you through required fields. Hit Calculate — results appear in under a second.
03
Export a Professional Report
One click generates an A4 PDF with letterhead, design basis, step-by-step workings, results summary, and an engineer sign-off block — submittal ready.

Quick Start

Running in under 2 minutes

Python 3.11+ and Git are the only requirements. Works on Linux, macOS, Windows, and Android (Termux).

1
Clone & install — one pip command, no virtualenv required
2
Start the API backend — FastAPI + uvicorn on port 8000
3
Start the UI — Streamlit on port 8501
4
Open browser — localhost:8501 for UI, localhost:8000/docs for API
View on GitHub ▶ Run in Colab
bash
# Clone the repository git clone https://github.com/kakarot-oncloud/openmep.git cd openmep # Install dependencies pip install -r requirements.txt # Terminal 1 — API backend uvicorn backend.main:app --reload --port 8000 # Terminal 2 — Streamlit UI streamlit run streamlit_app/app.py ✓ UI → http://localhost:8501 ✓ API → http://localhost:8000/docs

REST API

Every calculation is an endpoint

Interactive Swagger docs at /docs. All 35+ endpoints accept JSON and return structured results identically across all four regions.

curl — cable sizing · GCC / Dubai
curl -X POST http://localhost:8000/api/electrical/cable-sizing \ -H "Content-Type: application/json" \ -d '{ "region": "gcc", "sub_region": "dewa", "load_kw": 45, "power_factor": 0.85, "phases": 3, "cable_type": "XLPE_CU", "installation_method": "C", "cable_length_m": 80, "ambient_temp_c": 40 }' → { "selected_size_mm2": 16, "voltage_drop_pct": 4.28, "overall_compliant": true, "standard": "BS 7671" }
